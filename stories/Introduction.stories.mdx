# AI Playground Component Library

Welcome to the AI Playground component library! This Storybook contains all the reusable components used in the AI Playground interface.

## Design System

The AI Playground uses a glassmorphism design aesthetic with the following key features:

- **Color Palette**: Cyan and emerald theme with neutral grays
- **Typography**: Geist Sans for UI elements, Geist Mono for code
- **Glassmorphism Effects**: Backdrop blur and semi-transparent cards
- **Dark/Light Theme Support**: Seamless theme switching

## Core Components

### Interactive Elements
- **Button**: Various styles and sizes for different actions
- **Slider**: Parameter controls with tooltips and labels
- **Modal**: Dialogs for templates, confirmations, and settings

### Chat Interface
- **ChatBubble**: Message display with actions and metadata
- **Chat Interface**: Multi-tab conversation management
- **History Panel**: Searchable conversation history

### AI Controls
- **Model Selector**: Dropdown for AI model selection
- **Parameters Panel**: Sliders for AI model parameters
- **Prompt Editor**: Rich text editor with templates

## Usage

Each component is designed to be:
- **Accessible**: Proper ARIA labels and keyboard navigation
- **Responsive**: Works across mobile, tablet, and desktop
- **Themeable**: Supports light and dark modes
- **Reusable**: Consistent API and styling
